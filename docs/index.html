Basics

const Example = require('KasuNhentai'); // experimental, if it doesn't work use 'kasu.nhentaiapi.js'
    const ExampleApi = new Example();
    /**
     * or const ExampleApi = new Example('start', "https://nhentai.net");
     * If you don't want to add the url and connection.start() after it.
     * because it automatically start the connection.
     * 
     * Check the documentation for more info.
    */
    
    //          ----number | strings can do----
    const ID = 228922 //or "228922" or "https://nhentai.net/g/228922"
    async function json(){
    
        // start the connection
        ExampleApi.url = "https://nhentai.net"
        ExampleApi.connection.start()
    
        const val = await ExampleApi.getID(ID).json()
        val.url
        // OR
        ExampleApi.getID(ID).json(data=>{data.url})
        // OR
        (await ExampleApi.getID(ID).json()).url
        // result: https://nhentai.net/g/228922/
    
        // closes the connection
        ExampleApi.connection.close()
    }

⠀Constructor(connection, url, debug)⠀

connection has 1 inputs that can be accepted, start. Same as the connection function below. It just automatically start the connection without calling the connection property.

url see the url property below.

debug see the connection's debug below.

Property classes

⠀url⠀ : string

  • The Inputted URL will be used to connect. As of the 3.5.0 update the domain .to is now supported.
  • https://nhentai.net or https://nhentai.to the https:// must be included.

⠀connection⠀ : object

  • If you want to start the connection provided by the url.

    ⠀⠀- ⠀start( debug )⠀ : function

    ⠀⠀⠀⠀• start the connection.

    ⠀⠀- ⠀close( debug )⠀ : function

    ⠀⠀⠀⠀• close the connection.

The debug on the start, close functions is a boolean. If set to true It will show a log that shows that it is working and connected.

⠀IsDiscord⠀ : boolean

  • If you're using this API on your DiscordBot Enabling this will block some tags that is inappropriate or violates DiscordTOS.
  • default bool set is false.

⠀blockedWords⠀ : string

  • Add more tags that will be blocked. 
      blockedWords = "crossdressing brutality penetration"

⠀ReRollonFail⠀ : boolean

  • If you're using IsDiscord and don't wanna do your own retry function use this.
  • note: By enabling this it might slow down the process unless if you have a fast internet or processor.
  • default bool set is false.

⠀IgnoreNone⠀ : boolean

  • Ignore showing empty / none tags on the json object.
  • default bool set is false. 
    // false
    {
      "tag": "none",
      "parodies": "konosuba"
    }
    // true
    {
      "parodies": "konosuba"
    }

Method classes

⠀getID( string )⠀

  • string can be a string or a number or even the link itself. Example of this method can be seen on the Basics
  • returns a json object.

⠀pRandID()⠀

  • Generate random ID based on the website's button 'Random'.
  • returns an ID only!

⠀pRandSpecificTags( string, fn )⠀

  • It uses the Nhentai Searchbar functionality.
  • returns a json object. 
    //gets the data of the given tags
    await api.pRandSpecificTags("konosuba aqua sole female", data=>{
      /* data.[json object] */
    })

pRand's

    ⠀pRandTag( tag, fn )⠀

    ⠀pRandParody( tag, fn )⠀

    ⠀pRandArtist( tag, fn )⠀

    ⠀pRandGroup( tag, fn )⠀

  • Tag, Parody, Artist, Group shares the same function it generate random ID based on the Given Tag.
    NOTE: It only accepts 1 tag/name that exist on the Nhentai database.
  • returns a json object. 
    //returns a link | id
    await pRandtag("<name of the tag>").link | .id
    //get data of the random generated ID of the given tag
    await pRandtag("<name of the tag>", data=>{
      /* data.[json object] */
    })

⠀pRandom( fn )⠀

  • Generate random ID based on the website's button 'Random'.
  • returns a json object. 
    // gives random number
    await api.pRandom()
    // you can get the random number's data
    await api.pRandom(data=>{
      data.images.cover
    })

⠀pHomepage( int )⠀

  • int specifies the page. Shows the Homepage of the selected URL.
  • returns a json object

⠀pLanguagePage( str, int )⠀

  • str only accepts 4 string japanese, english, chinese and translated (is this even a language). it also accepts shortcuts like jp, japan, eng, china. int specifies the page.
  • returns a json object

⠀pSearch( str, int )⠀

  • Shows some list of the searched string of str. int specifies the page.
  • returns a json object

⠀pTagPage( str, int )⠀

  • Shows some list of the searched tag string of str. int specifies the page.
  • returns a json object
